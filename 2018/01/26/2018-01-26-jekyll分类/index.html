jekyll 列出某一分类下的所有文章

  • hack
  • 2018-01-26

有一个笨办法可以累出blog下某一分类的所有文章

1、新建一个文件夹

2、在文件中新建一个html，例如ANSYS.html

{% for post in site.categories.ANSYS %} 
    

    

    
{{ post.title }}{{ post.date | date: "%B %-d, %Y" }}

    
{{post.tag}}

    

    

    
{{ post.excerpt }}

    

    

    {% endfor %}

第一行ANSYS代表你的categories

在头文件中计入路径

permalink: /ansys/

在导航栏中链接到该html

<ul class="nav nav-pills">
<li role="presentation" class="active"><a href="{{site.baseurl}}/">梦里寻欢</a></li>
<li role="presentation"><a href="/ansys">ANSYS</a></li>
</ul>
