opencv 安装配置

  • hack
  • 2018-03-10

运行环境：Windows10+vs17 OpenCV版本3.4.1

1 下载OpenCV https://opencv.org/releases.html 目前版本是3.4.1 并解压 我的解压位置为D:\Install\opencv

2 启动vs2017 创建一个c++控制台应用程序

3 在项目右键 打开属性,首先确认当前配置

X86和x64 release和debug 这个需要手动指定 默认一般是x86，如果你在release+x86下配置了半天，却在debug+x64下调试 会出错的
Vc++目录 - 包含目录添加D:\Install\opencv\build\include路径下的文件夹

引用目录和库目录添加D:\Install\opencv\build\x64\vc15路径下的文件夹

连接器-输入-附加依赖项将D:\Install\opencv\build\x64\vc15\lib路径下的opencv_world341.lib和opencv_world341d.lib两个文件名赋值复制过去

注意：如果配置release的话 只需要复制opencv_world341.lib 而在debug下复制opencv_world341d.lib

如果在运行的时候提示opencv_world341.dll和opencv_world341d.dll丢失，一个暴力解决办法是将D:\Install\opencv\build\x64\vc15\bin路径下的opencv_world341.dll和opencv_world341d.dl复制到C:\Windows\System32

一切就绪 开始调试程序

#include "stdafx.h"
#include <opencv2/opencv.hpp>
#include <iostream>
using namespace std;
using namespace cv;
int main()
{
    Mat image = imread("D:\\x2.png");  //存放自己图像的路径 
    imshow("显示图像", image);
    waitKey(1110);
    return 0;
}

需要注意这一行

Mat image = imread("D:\\x2.png");

路径是需要加’\‘的
如果正确无误，将图片读取并显示
